Сборка и запуск под Linux через Wine

Этот репозиторий содержит скрипт `setup_and_build_masm32.sh`, который автоматизирует установку Wine (через apt), распаковку MASM32 в Wine-префикс и запуск `build.bat` для сборки `main.exe`.

Важные замечания:
- Скрипт рассчитан на Debian/Ubuntu-подобные системы с apt.
- Требуется доступ sudo для установки пакетов.
- Некоторые окружения (контейнеры, ограниченные CI-системы) могут блокировать установку Wine или запуск GUI-программ.

Как запустить (на вашей локальной машине):

```bash
chmod +x setup_and_build_masm32.sh
sudo ./setup_and_build_masm32.sh
```

После успешной сборки `main.exe` будет скопирован в `build_output/main.exe`.

Если что-то пойдёт не так — пришлите вывод команды и я помогу с диагностикой.
