Краткие инструкции для сборки и запуска на Windows

Требования:
- Установленный MASM32 (распаковать masm32v11r.zip в C:\masm32)

Инструкция:
1) Откройте командную строку (cmd.exe) с правами пользователя.
2) Перейдите в папку репозитория, содержащую `main.asm`.
3) Запустите `build.bat`:

```bat
build.bat
```

Если сборка прошла успешно, появится `main.exe`.

Примечание: текущая конфигурация `.devcontainer/devcontainer.json` использует Windows Server Core, поэтому для локального запуска devcontainer необходим хост с поддержкой Windows-контейнеров (Docker Desktop на Windows).
